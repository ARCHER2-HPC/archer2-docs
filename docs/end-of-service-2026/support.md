# Getting help with ARCHER2 end of service

These documentation pages will be updated with information as it 
becomes available. The last update date of the docs will be published
on the top level page of this documentation section so you can assess 
if anything has changed since you last looked.

If you have any questions around the end of service, you can contact
the [ARCHER2 service desk](https://www.archer2.ac.uk/support-access/servicedesk.html)
and we will do our best to answer your questions.
